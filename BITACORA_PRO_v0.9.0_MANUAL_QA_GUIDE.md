## Quick Start

This guide provides step-by-step instructions for testing all v0.9.0 features.

### Prerequisites
- Android device or emulator (API 26+)
- APK built from latest code
- Sample WhatsApp chat export (.txt file)

---

## Test Flows

### 1. Version Display Verification ✅

**Objective**: Verify v0.9.0 is displayed across all screens

**Steps**:
1. Launch app
2. Check WelcomeScreen - should show "v0.9.0 - Product Polish & Assistant"
3. Tap "Continuar"
4. Check HomeScreen footer - should show "Bitacora Pro v0.9.0"
5. Create a job and generate PDF report
6. Check PDF footer - should show "Generated by Bitacora Pro v0.9.0"

**Expected Results**:
- ✓ WelcomeScreen shows v0.9.0
- ✓ HomeScreen footer shows v0.9.0
- ✓ PDF footer shows v0.9.0

---

### 2. About Screen Access & Display ✅

**Objective**: Verify AboutScreen is accessible and displays correct information

**Steps**:
1. From HomeScreen, tap the ⋮ (three dots) menu icon
2. Tap "Acerca de"
3. Verify all sections are displayed:
   - App name: "Bitacora Pro"
   - Version: "Versión: v0.9.0"
   - Build type: "Tipo de compilación: Pilot"
   - Privacy section with statement
   - Assistant section showing:
     - ✓ IA local activa
     - ✗ IA en línea desactivada
   - Permissions section with explanations
   - WhatsApp disclaimer
4. Tap back arrow to return to HomeScreen

**Expected Results**:
- ✓ AboutScreen opens without errors
- ✓ All sections display correctly
- ✓ Back button returns to HomeScreen
- ✓ Text is readable and properly formatted

---

### 3. File Menu Navigation ✅

**Objective**: Verify File Menu opens and all options are visible

**Steps**:
1. From HomeScreen, tap the ⋮ (three dots) menu icon
2. Verify all 6 menu items are visible:
   - Importar chat de WhatsApp
   - Importar imágenes
   - Importar PDF o documento
   - Importar texto
   - Exportar reporte PDF
   - Acerca de
3. Tap elsewhere to close menu

**Expected Results**:
- ✓ Menu opens without errors
- ✓ All 6 items are visible
- ✓ Menu closes when tapping elsewhere
- ✓ Menu closes when selecting an item

---

### 4. WhatsApp Chat Import Flow ✅

**Objective**: Verify WhatsApp chat import works correctly

**Prerequisites**:
- Have a WhatsApp chat export (.txt file) on device
- To export: WhatsApp → Chat → Long press chat → More → Export chat → Without media

**Steps**:
1. From HomeScreen, tap ⋮ menu
2. Tap "Importar chat de WhatsApp"
3. File picker opens
4. Navigate to and select a .txt WhatsApp export file
5. Verify chat is imported:
   - Go to "Sin asignar" (Inbox)
   - New text evidence should appear with chat summary
   - Summary should show:
     - "WhatsApp Chat Export"
     - Message count
     - Participant count
     - Extracted phone numbers (if any)
     - Chat preview (first 5000 chars)
6. Tap the imported chat to view full content
7. Optionally assign to an activity

**Expected Results**:
- ✓ File picker opens with text/plain filter
- ✓ Chat file is read successfully
- ✓ Chat appears in inbox as text evidence
- ✓ Summary includes all expected information
- ✓ Chat can be assigned to activities
- ✓ No crashes or errors

---

### 5. Ask Screen & LocalAssistantProvider ✅

**Objective**: Verify Ask screen works and LocalAssistantProvider answers questions

**Steps**:
1. From HomeScreen, tap "Asistente" (Assistant) quick action
2. Verify AssistantScreen shows 5 action cards:
   - Revisar Faltantes
   - Capturar Evidencia
   - Preparar Reporte
   - Cerrar Actividad
   - **Pregunta al Asistente** (NEW)
3. Tap "Pregunta al Asistente"
4. AskScreen opens with:
   - Title: "Pregúntale a Bitacora"
   - Input field with hint: "¿Qué necesitas saber?"
   - Offline mode indicator
5. Ask a question, e.g., "¿Cómo capturar evidencia?"
6. Verify response appears in chat
7. Ask another question to test conversation history
8. Test various question types:
   - "¿Cómo capturar evidencia?"
   - "¿Qué es privacidad?"
   - "¿Cómo completar una actividad?"
   - "¿Qué es WhatsApp?"
9. Tap back arrow to return to AssistantScreen

**Expected Results**:
- ✓ AssistantScreen shows 5 action cards
- ✓ "Pregunta al Asistente" card is visible
- ✓ AskScreen opens without errors
- ✓ Questions are answered by LocalAssistantProvider
- ✓ Answers are relevant and helpful
- ✓ Chat history is maintained
- ✓ Offline mode indicator is visible
- ✓ Back button returns to AssistantScreen

---

**Steps**:
1. From HomeScreen, tap "Asistente"

7. Go back to AssistantScreen
8. Tap "Cerrar Actividad" → Should navigate to HomeScreen
9. Go back to AssistantScreen
10. Tap "Pregunta al Asistente" → Should navigate to AskScreen

**Expected Results**:
- ✓ All 5 action cards navigate correctly
- ✓ No crashes or errors
- ✓ Back button works from all destinations
